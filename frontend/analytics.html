City Size Tiers

Quantile-based urban hierarchy classification

Doughnut
▶ Python — Logic 
# ── Define thresholds from actual data ──
q75 = df['Population (Est.)'].quantile(0.75)
q25 = df['Population (Est.)'].quantile(0.25)
print(f"Mega threshold: ≥ {q75/1e6:.1f}M")
print(f"Large threshold: ≥ {q25/1e6:.1f}M")

# ── Classify ──
def classify(pop):
    if pop >= q75: return 'Mega City'
    if pop >= q25: return 'Large City'
    return 'Medium City'

df['Size Category'] = df['Population (Est.)'].apply(classify)

# ── Aggregate ──
size_dist = df['Size Category'].value_counts().to_dict()
Insight: Using quantile-based thresholds (not arbitrary cut-offs) ensures balanced classes: 25% Mega, 50% Large, 25% Medium. This avoids the common pitfall of skewed class distributions.

Area vs Population Correlation

Bivariate analysis — does bigger land area mean more people?

Scatter
▶ Python — NumPy / SciPy 
# ── Extract both variables ──
X = df['Area (sq km)'].values
Y = df['Population (Est.)'].values

# ── Pearson correlation ──
from scipy.stats import pearsonr
r, p_val = pearsonr(X, Y)
print(f"Pearson r = {r:.4f}  (p = {p_val:.2e})")

# ── Build scatter payload ──
scatter = [
    {'x': float(x), 'y': float(y)}
    for x, y in zip(X, Y)
]

# ── Interpretation ──
if abs(r) < 0.3:
    print("→ Weak correlation: area alone is a poor predictor")
Insight: The Pearson correlation is typically weak (r ≈ 0.2–0.4), proving that land area alone cannot explain population. Dense cities like Dhaka achieve massive populations on compact footprints, while sprawling cities like Kinshasa occupy vast areas with lower density.

Population Distribution

Frequency across equal-width bins — shape of the distribution

Histogram
▶ Python — NumPy 
# ── Raw population array ──
pop = df['Population (Est.)'].values

# ── Create 10 equal-width bins ──
bins = np.linspace(pop.min(), pop.max(), 11)
counts, edges = np.histogram(pop, bins=bins)

# ── Format bin labels ──
labels = [
    f"{edges[i]/1e6:.1f}M–{edges[i+1]/1e6:.1f}M"
    for i in range(len(counts))
]

# ── Identify modal bin ──
peak = np.argmax(counts)
print(f"Peak bin: {labels[peak]} ({counts[peak]} cities)")
Insight: The right-skewed shape is characteristic of a log-normal distribution — a small number of extreme megacities pull the tail right, while the bulk of cities cluster in lower population ranges.
